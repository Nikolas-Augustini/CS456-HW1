# CS456-HW1
This is a group assignment for CS456 at CSU. The main objective is to get familiar with the basics of VR through Unity.
Below is the assignment specifications:
A. Controllers and Hands Setup using XR Interaction Tool Kit: (10 points)

a. You can either download the provided VR template or create your own using Unity.
b. Please ensure that your VR headset is successfully connected to your PC or laptop.
c. When running the project, please complete the setup for the controller and hand tracking.

B. Controllers and Hands Setup using Meta XR SDK: (10 points)

a. You can either download the provided VR template or create your own using Unity.
b. Please ensure that your VR headset is successfully connected to your PC or laptop.
c. When running the project, please complete the setup for the controller and hand tracking

C. Assignment Task 1 (Total 10 Points):

This task focuses on manipulation of the object within the VR environment using both hands and the controllers. Students will create a platform and create various small objects mentioned below and will interact with them. After completing this task students will understand the basics of object interaction within VR environments. The requirements are segregated to different groups, so keep that in mind while you are completing your tasks.

Platform and Object creations using XR Interaction Toolkit (Total 5 Points)

1. Platform Creation: (2.5 Points
a. A platform (in the form of a table) should be created within the VR environment. (Requirement for Group of 1, 2, 3, and 4)

2. Object Creation: (2.5 Points)
a. Red Block: A red block should be created and placed on the platform, ensuring that it can be grabbed using a hand or controller. (Requirement for Group of 1, 2, 3, and 4)
b. Blue Block: A blue block should be created and placed on the platform. This block should not be grabbable; however, when a hand or controller is placed near it, its color should change from blue to white. (Requirement for Group of 1, 2, 3, and 4)
c. Yellow Block: A yellow block should be created and placed on the platform. This block should not be grabbable; however, when a hand or controller is placed near it, a sound should be produced. (Requirement for Group of 2, 3, and 4)
d. Purple Block: A purple block should be created and placed on the platform. This block should not be grabbable; however, when a hand or controller is placed near it, the block should disappear. (Requirement for Group of 3 and 4)

D. Assignment Task 2 (Total 20 Points):

This task focused on gesture recognition using only your hands. In this task, students will do a hand gesture, and that will be displayed within the VR environment; specific instructions are provided below. After completing this task, students will understand how to implement gestures within VR environments. The requirements are segregated to different groups, so keep that in mind while you are completing your tasks. This part of assignment task must be done using both XR Interaction Toolkit kit, and Meta XR SDK

Gesture Recognition Implementation using XR Interaction Tool Kit (10 Points):

a. Thumbs Up: When a thumbs-up gesture is made with a hand, the VR system should recognize the gesture and display a message in the environment indicating either “Right hand thumbs up!” or “Left hand thumbs up!” based on the hand used. (Requirement for Group of 1, 2, 3, 4)
b. Fist: When a fist gesture is made, the VR system should recognize the gesture and display a message indicating either “Right hand fist!” or “Left hand fist!” based on the hand used. (Requirement for Group of 2, 3, 4)
c. Custom Gesture (e.g., Peace Sign): A custom gesture, such as a peace sign, should be implemented. The VR system should recognize the gesture and display a message indicating either “Right-hand Peace Sign!” or “Left-hand Peace Sign!” based on the hand used. (Requirement for Group of 3 and 4)
